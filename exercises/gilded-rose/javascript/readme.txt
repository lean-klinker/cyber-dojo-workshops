What are we building?

A calculator that takes a string of numbers and calculates the sum of the numbers.

REQUIREMENTS:

1. When adding an empty string then returns 0

__________________________________________________________

2. When adding two numbers separated by a comma then returns the sum of the two numbers

__________________________________________________________

3. When adding multiple numbers separated by commas then returns the sum of all numbers

__________________________________________________________

4. When adding multiple numbers separated by new-lines then returns the sum of all numbers

__________________________________________________________

5. When adding multiple numbers separated by new-lines or commas then returns the sum of all numbers

__________________________________________________________

6. When adding numbers containing a custom delimiter then returns the sum of all numbers

    6a. Custom delimiters are indicated using "//[delimiter]": 
        6a1. Example: "//[;]1;2;3"

__________________________________________________________

7. When adding numbers containing a negative number then throws negatives not allowed

__________________________________________________________

8. When adding multiple negative numbers then throws negatives not allowed with each negative number in the message

__________________________________________________________

9. When adding with numbers greater than 1000 then ignores numbers greater than 1000

    9a. Example: "1000,2" = 2

__________________________________________________________

10. When adding numbers containing a multi-character custom delimiter then returns the sum of all numbers

    10a. Example: "//[*;]12*;5*;9"

__________________________________________________________

11. When adding numbers containing multiple custom delimiters then returns the sum of all numbers

__________________________________________________________

12. When adding numbers containing multiple multi-character custom delimiters then returns the sum of all numbers
